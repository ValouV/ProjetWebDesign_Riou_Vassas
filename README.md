# projet-web-design

Instruction pour ouvrir l'application WEB :
-Télécharger le dossier
-Ouvrir un terminal (powershell recommandé sur windows)
-Aller à la racine du dossier : taper : cd chemin_ou_est_le_dossier
-Installer NPM : taper : npm install
-Installer Bower : taper : npm install -g bower
-Installer les dépendances via bower : taper : bower install
-Lancer l'application en local en tapant : grunt serve

Fonctionnalités implémentées :
-AngularJS utilisé, code indenté et commenté
-Utilisation d'un webservice externe pour avoir des données
-Utilisation du format JSON pour ces données
-Utilisation des directives core
-Site web personnalisé et unique 
-Site web respectant les principes de material design
-Site web responsive
-Filtre custom pour afficher 1M6 plutot que 1600000 dans la population des villes

Fonctionnalités supplémentaires :
-Utilisation de mapboxgl afin d'avoir une carte zoomable et d'y afficher des points sur les villes les plus peuplées ces points étant plus ou moins gros selon la popoluation de la ville
